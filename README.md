 Movie Search App

A sleek, modern web app to search for movies and get full details using the [OMDb API](https://www.omdbapi.com/).  
Built with **HTML, CSS, Bootstrap, and JavaScript**, this app features a clean dark UI and responsive layout.


  Features

- Search for any movie by name
- Displays movie poster, title, release date, writer, actors, language, country, awards, IMDb rating, and plot
- Dark cinematic UI for a professional look
- Responsive design for mobile and desktop
- Error handling for movies not found
- Simple and lightweight — runs entirely in the browser



 Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Fetch API)
- [OMDb API](https://www.omdbapi.com/)
